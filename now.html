Listening

Podcast Queue

Your Podlink collection or favorite podcast episodes will appear here.

How to add your podcasts:

  1. Set up your Podlink account at podlink.to
  2. Add episodes you're listening to
  3. Get the embed code or shareable link
  4. Paste it here

Alternative: Manually list episodes with links to Apple Podcasts, Spotify, etc.

Pinning

Pinterest Board

Your curated Pinterest board will appear here.

How to embed your Pinterest board:

  1. Create a board on Pinterest for this site
  2. Go to the board
  3. Click the three dots (...)
  4. Select "Get board widget"
  5. Copy the embed code
  6. Paste it here

Or use Pinterest's Widget Builder: widgets.pinterest.com
